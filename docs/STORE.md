## Чеклист

- [ ] `npm run zip` → загрузить `tab-memory-extension.zip`
- [ ] Сайт задеплоен на Vercel, privacy доступен по HTTPS
- [ ] Скриншоты: 1280×800 или 640×400 (popup + страница истории)
- [ ] Промо: маленький тайл 440×280, по желанию — marquee 1400×560
- [ ] Категория: **Productivity**
- [ ] Указать: данные только локально, без аккаунта и серверов

## Текст для карточки (RU)

**Краткое описание:** Сохраняйте вкладки со скриншотом и заметкой, закрывайте их — контекст не потеряется.

**Подробное описание:** Tab Memory помогает убрать лишние вкладки, не забыв, зачем вы их открывали. Нажмите на иконку, добавьте заметку и сохраните: расширение сделает скриншот страницы, запомнит URL и заголовок, закроет вкладку. Вся история хранится только на вашем компьютере. Откройте историю, чтобы снова перейти на сайт или удалить старые записи.

## Обоснование разрешений

| Разрешение | Зачем |
|------------|--------|
| activeTab | Скриншот по клику пользователя |
| tabs | Закрытие вкладки, открытие истории и сохранённых URL |
| &lt;all_urls&gt; | Снимок и повторное открытие любых сайтов, которые вы сами сохраняете |
| unlimitedStorage | Локальное хранение скриншотов в IndexedDB |
