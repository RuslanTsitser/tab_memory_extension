Tab Memory делает скриншот, сохраняет заметку, URL и заголовок страницы — и закрывает вкладку. Больше не нужно копить табы или забывать, зачем добавляли закладку.

Проблема лишних вкладок

Хочется навести порядок в браузере, но страшно закрыть — вдруг забудете, зачем открывали. Обычные закладки тоже копятся без контекста.

Без Tab Memory

  • Десятки вкладок мешают работать
  • Закрыл — потерял контекст навсегда
  • Закладки без «зачем» бесполезны

С Tab Memory

  • Сохранить и закрыть в один клик
  • Скриншот + заметка = полный контекст
  • Вернуться через локальную историю

Как это работает

Три секунды — и вкладка сохранена с полным контекстом.

Откройте popup

Нажмите на иконку Tab Memory на нужной вкладке.

Добавьте заметку

Кратко опишите, зачем открыли — пригодится позже.

Сохраните и закройте

Скриншот и метаданные сохранены, вкладка закрыта.

Вернитесь из истории

Откройте History, найдите превью, кликните — URL откроется в новой вкладке.

FAQ

Синхронизация между устройствами?

Нет. Всё хранится локально в браузере — без облака и аккаунта.

Какие данные сохраняются?

Скриншот (JPEG), URL, заголовок, заметка и время. Ничего не отправляется на сервер.

Можно сохранить chrome:// страницы?

Нет — системные страницы браузера нельзя снимать по соображениям безопасности.

Как удалить всё?

«Очистить историю» на странице History или удаление расширения.

Данные остаются на вашем устройстве

Tab Memory ничего не собирает: без аналитики, аккаунтов и передачи третьим лицам.
